## üì¶ Installation & Setup

To run these notebooks locally, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/Basics_python.git
   cd Basics_python
   ```

2. **Install Jupyter (if not already installed):**
   Using Pip:
   ```bash
   pip install notebook
   ```
   Using Conda:
   ```bash
   conda install -c conda-forge notebook
   ```

3. **Launch the Notebook server:**
   ```bash
   jupyter notebook
   ```
